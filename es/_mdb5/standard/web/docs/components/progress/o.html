Documentación y ejemplos para el uso de barras de progreso personalizadas con soporte para barras apiladas, fondos animados y etiquetas de texto.

Ejemplo básico

Cómo funciona

Los componentes de progreso se compilan con dos elementos HTML, algo de CSS para establecer el ancho y algunos atributos. No usamos el HTML5 <progress> elemento, asegurándose de que puede apilar barras de progreso, animarlas y colocar etiquetas de texto sobre ellas.

  • Usamos .progress como envoltorio para indicar el valor máximo del progreso bar.
  • Usamos el .progress-bar interno para indicar el progreso hasta ahora.
  • .progress-bar requiere un estilo en línea, una clase de utilidad o CSS personalizado para configurar su ancho.
  • .progress-bar también requiere algún rol y aria atributos para que sea accesible.

Pon todo eso junto y tienes los siguientes ejemplos.

MDB proporciona un puñado de utilidades para configurar el ancho . Depende de tu necesidades, estos pueden ayudar a configurar rápidamente el progreso.

Etiquetas

Agregue etiquetas a sus barras de progreso colocando texto dentro del .progress-bar .

Para que la etiqueta sea visible, debe establecer una altura adecuada para la barra.

Altura

Solo establecemos un valor de height en .progress , así que si lo cambia valor el .progress-bar interno cambiará de tamaño automáticamente en consecuencia.

Colores

Utilice clases de utilidad en segundo plano para cambiar la apariencia de las barras de progreso individuales.

Varias barras

Incluya varias barras de progreso en un componente de progreso si es necesario.

A rayas

Agregue .progress-bar-striped a cualquier .progress-bar para aplicar una banda a través de Gradiente CSS sobre el color de fondo de la barra de progreso.

Rayas animadas

El degradado de rayas también se puede animar. Añadir .progress-bar-animated a .progress-bar para animar las franjas a la derecha a la izquierda a través de animaciones CSS3.
