Los grupos de listas son un componente flexible y poderoso para mostrar una serie de contenido. Modificar y ampliarlos para que admitan prácticamente cualquier contenido.

Nota: lea la pestaña API para encontrar todas las opciones disponibles y personalización avanzada

Separamos estas pseudoclases para asegurar que los grupos de listas hecho de elementos no interactivos (como <li>s o <div>s) no proporcione una prestación de clic o toque.

Asegúrese de no utilizar las clases .btn estándar aquí .

Horizontal

Agregue .list-group-horizontal para cambiar el diseño de los elementos del grupo de lista de vertical a horizontal en todos los puntos de interrupción. Alternativamente, elija una variante receptiva .list-group-horizontal- {sm | md | lg | xl | xxl} para hacer que un grupo de listas comience horizontal en el ancho mínimo de ese punto de interrupción. Actualmente Los grupos de listas horizontales no se pueden combinar con grupos de listas alineadas.

ProTip: ¿Desea elementos de grupo de lista de igual ancho en horizontal? Añadir .flex-fill a cada elemento del grupo de lista.

Transmitir significado a las tecnologías de asistencia:
El uso del color para agregar significado solo proporciona una indicación visual, que no se transmitirá a usuarios de tecnologías de asistencia, como lectores de pantalla. Asegúrese de que la información indicada por el color es obvio por el contenido mismo (por ejemplo, el texto visible), o está incluido a través de medios alternativos, como texto adicional oculto con el Clase .sr-only .
