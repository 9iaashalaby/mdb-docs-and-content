Métodos

Puede crear una instancia de botón con el constructor de botón, por ejemplo:

var button = document.getElementById('myButton') var bsButton = new bootstrap.Button(button)
Método Descripción
palanca Alterna el estado de empuje. Da al botón la apariencia de que ha sido activado.
disponer Destruye el botón de un elemento.

Por ejemplo, para alternar todos los botones

var buttons = document.querySelectorAll('.btn') buttons.forEach(function (button) { var button = new bootstrap.Button(button) button.toggle() })
