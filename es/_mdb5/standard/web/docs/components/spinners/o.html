Se pueden utilizar “hilanderos” MDB para mostrar el estado de carga en sus proyectos. Están construidos solo con HTML y CSS, lo que significa que no necesita JavaScript para crearlos. Sin embargo, necesitará algunos JavaScript personalizados para cambiar su visibilidad. Su apariencia, alineación y tamaño pueden personalizarse fácilmente con nuestras increíbles clases de servicios públicos.

Por motivos de accesibilidad, cada cargador aquí incluye role = "status" y un anidado <span class="sr-only">Loading...</span>.

Colors

La ruleta de bordes usa currentColor para su border-color , lo que significa puedes personalizar el color con utilidades de color de texto . Puede utilizar cualquiera de nuestras utilidades de color de texto en la ruleta estándar.

Los spinners en Bootstrap se construyen con rem s, currentColor y display: inline-flex . Esto significa que se pueden cambiar de tamaño, cambiar de color y alineado rápidamente.
