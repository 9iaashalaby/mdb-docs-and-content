MDB cuenta con el respaldo de un extenso sistema de colores de Material Design que inspira nuestros estilos y componentes. Esto permite una personalización y una extensión más completas para cualquier proyecto.

Tema

Usamos un subconjunto de todos los colores para crear una paleta de colores más pequeña para generar esquemas de color, también disponible como variables Sass y un mapa Sass en Bootstrap Archivo scss / free / _variables.scss .

Todos estos colores están disponibles como un mapa Sass, $ theme-colors en Archivo scss / free / _variables.scss .

$theme-colors: ( "primary": $primary, "secondary": $secondary, "success": $success, "info": $info, "warning": $warning, "danger": $danger, "light": $light, "dark": $dark );

Nota: los colores del tema también están disponibles como clases CSS. Puedes usarlos directamente en su proyecto sin compilar código scss. Eche un vistazo a los ejemplos a continuación.

Componentes

También usamos colores de tema como colores predefinidos para algunos componentes, por ejemplo, botones.

Paleta completa

Todos los colores MDB están disponibles como variables Sass y un mapa Sass en Archivo scss / free / _variables.scss . Para evitar mayores tamaños de archivo, no creamos clases de color de texto o de fondo para cada una de estas variables. En cambio, elegimos un subconjunto de estos colores para una paleta de temas .

Asegúrese de controlar las relaciones de contraste al personalizar los colores. Como se muestra a continuación, agregamos tres proporciones de contraste para cada uno de los colores principales: uno para los colores actuales de la muestra, uno para contra blanco y uno a favor contra negro.
